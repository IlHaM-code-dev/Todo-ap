# My First To-Do App ✅

This is a simple **To-Do List web app** built entirely on mobile using **HTML, CSS, and JavaScript** via Acode app.

## 📋 Features
- Add tasks
- Simple & clean UI
- Works offline (no internet needed)

## 🚀 How to Use
1. Download the code
2. Open `index.html` in any browser
3. Start adding your tasks

## 📁 Files Included
- `index.html`
- `style.css`
- `script.js`
